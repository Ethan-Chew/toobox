# Toobox
Toobox of different tools to help in your Academics

## Required Libraries 
### Chemlib
1. Open the terminal and move to the project directory
2. Run ```pip3 install chemlib```

## Current Tools
1. Ionic Equation Solver
2. Quadratic Equation Solver
3. Matrix Solver
